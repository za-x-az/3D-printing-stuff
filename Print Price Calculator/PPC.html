📖 Help Guide

🧮 Calculator

  • Filaments: 3 rows load by default. Pick Type first — the Brand dropdown auto-filters to matching brands. Enter grams used per job.
  • Printer & Time: Select a printer; all costs derive automatically from its lifecycle settings. Enter hours and minutes.
  • Hardware: "Add Hardware" opens a searchable modal. Quantity is per printed item.
  • Job Settings: Set Items per Job and Profit Margin %. Global values (VAT, Setup Fee, Filament Markup) are pulled from Settings.

⚙️ Global Settings

  • VAT % — applied on top of the sale price.
  • Power Cost (kWh) — your electricity rate.
  • Setup Fee — flat fee added to every job, excluded from the profit markup base.
  • Filament Markup % — global uplift on every filament line's raw material cost.
  • Mass Delete — permanently wipes all DB data. Export a backup first!

🖨️ Printer Hourly Cost Formula

With Payoff Time: Hourly Cost = (Machine Cost + Maintenance × Payoff Time) ÷ Expected Life Hours Without Payoff Time: Hourly Cost = Machine Cost ÷ Expected Life Hours Printer Job Cost = Print Time (hrs) × Hourly Cost

Payoff Time is optional (e.g. 0.5 = 6 months, 1 = 1 year). When left blank, only the Machine Cost is amortised over the Expected Life Hours. Maintenance is the annual upkeep cost.

🎨 Filament Cost Formula

Filament Cost = Grams Used × (Spool Price / Spool Weight g) × (1 + Filament Markup %)

Each filament carries its own spool weight so a 750 g spool is priced correctly vs a 1 kg spool.

⚡ Power Cost Formula

Power Cost = Print Hours × (Printer Watts / 1000) × Power Cost per kWh

🔩 Hardware Cost

Hardware Total = Unit Price × Quantity per Item × Items per Job

Hardware is passed through at cost — no profit markup applied.

💰 Profit & Pricing

Cost = Filament + Printer + Power + Hardware + Setup Fee Profit Base = Filament + Printer + Power + Hardware ← Setup Fee excluded Profit = Profit Base × Profit Margin % Sale (ex VAT) = Cost + Profit Sale (inc VAT) = Sale (ex VAT) × (1 + VAT %)

📤 CSV Import / Export

  • Each database view has Export CSV and Import CSV.
  • Export downloads all current items as a clean CSV with a header row.
  • Import matches by name (printers/hardware) or brand+type (filaments) and upserts — existing records update, new ones are added. Headers are case-insensitive; column order doesn't matter.

💾 Save & Download

The green button in the sidebar generates a new self-contained HTML file with all DB data and settings baked in. The version number auto-increments each time (e.g. calculator_v2.html).
